# Movi — Correcciones y Ajustes del DS

> Archivo vivo. Se actualiza durante el desarrollo con correcciones al DS, decisiones revisadas, y ajustes basados en feedback real.

---

## Cómo usar este archivo

Cuando algo del DS no funcione en la práctica — un color que no se ve bien en un contexto específico, un token de motion que resulta demasiado lento en dispositivos de gama baja, un patrón de copy que el usuario malinterpreta — se documenta aquí antes de actualizar el archivo correspondiente.

**Formato de cada corrección:**

```
## [FECHA] — [ÁREA] — [TÍTULO BREVE]
**Problema detectado:** Qué no funcionó y dónde
**Contexto:** En qué pantalla/componente/momento ocurrió
**Corrección:** Qué se cambió
**Archivos actualizados:** Lista de docs modificados
```
